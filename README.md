# UNIVERSITY MANAGEMENT SYSTEM 
A simple command-line system for managing university students, professors, and courses.

###Features:

###  STUDENTS
- Register for courses (up to 7)
- View their grades
- Track attendance
- View GPA

###  PROFESSORS
- Mark attendance
- Assign grades
- View course students

###  ADMIN
- Add/remove students
- Add/remove professors
- Add/remove courses
